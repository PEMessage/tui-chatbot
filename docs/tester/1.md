## 发现的 Bug（已修复）

在测试过程中发现并修复了以下问题：

1. **测试语法错误** (`test_event_stream.py`)
   - 问题: `async def async def test_result_without_iteration` 重复关键字
   - 修复: 移除重复的 `async def`

2. **测试逻辑错误** (`test_core_events.py`)
   - 问题: `test_elapsed_property` 和 `test_tps_property` 未调用 `finalize()`
   - 修复: 添加 `stats.finalize()` 调用

3. **测试异步问题** (`test_abort_controller.py`)
   - 问题: `test_repr_with_timeout` 在未运行事件循环时检查 `has_timeout`
   - 修复: 改为异步测试，允许时间让超时定时器设置

### ✅ B3: TokenStats.elapsed 修复

**位置**: `src/tui_chatbot/core/events.py` 第212-217行

**实现代码**:
```python
@property
def elapsed(self) -> float:
    """计算耗时，如果未结束返回 0.0."""
    if self.start_time is None or self.end_time is None:
        return 0.0
    return max(0.0, self.end_time - self.start_time)
```

**测试验证**: `test_elapsed_zero_when_no_end` ✅ 通过

**评审意见**: 正确修复，语义清晰：只有完成的统计才返回非零耗时。

### 3.2 ✅ 新增内置工具

**文件**: `src/tui_chatbot/agent/tool.py`

#### CalculatorTool (计算器)

**安全评估**: ✅ 安全
- 使用受限的 `eval()`，仅允许数学函数
- `__builtins__` 设为空字典
- 仅暴露白名单函数

```python
allowed_names = {
    "abs": abs, "max": max, "min": min,
    "sin": lambda x: __import__("math").sin(x),
    "cos": lambda x: __import__("math").cos(x),
    # ... 其他数学函数
}
result = eval(params.expression, {"__builtins__": {}}, allowed_names)
```

#### ReadFileTool (文件读取)

**安全评估**: ✅ 安全

- 检查文件存在性和类型
- 限制最大读取行数（默认 1000）
- 使用 UTF-8 编码，错误时替换

#### WriteFileTool (文件写入)

**安全评估**: ✅ 安全
- 自动创建父目录

- 使用 UTF-8 编码

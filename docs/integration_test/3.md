### 1.5 Provider 注册验证

**验证代码** (registry.py 第332-345行):
```python
# 尝试注册 Ollama (本地模型) - 使用延迟加载
ollama_base_url = os.getenv("OLLAMA_BASE_URL", "http://localhost:11434")

async def _load_ollama():
    from .ollama_provider import OllamaProvider, OllamaProviderConfig
    config = OllamaProviderConfig(base_url=ollama_base_url)
    return OllamaProvider(config)

# 延迟注册 OllamaProvider
ProviderRegistry.register_lazy(
    "ollama",
    _load_ollama,
    name_hint="ollama",
)
```

**结果**: ✅ **通过**
- OllamaProvider 已在注册表中延迟注册
- 使用 `register_lazy` 模式避免启动时导入 heavy 依赖
- 当 Ollama 服务不可用时不会导致启动失败

### 2.2 新增工具验证

#### CalculatorTool
- **名称**: calculate
- **功能**: 数学表达式计算
- **支持运算**: +, -, *, /, **, sin, cos, tan, sqrt, log, log10, exp, abs, max, min, sum, pow, round
- **常量**: pi, e
- **安全性**: 使用受限 eval，只允许数学函数和常量

#### ReadFileTool
- **名称**: read_file
- **功能**: 读取文本文件内容
- **参数**: file_path (路径), limit (最大行数，默认 1000)
- **特性**: 支持 ~ 扩展，自动处理编码错误，行数限制防止大文件

#### WriteFileTool
- **名称**: write_file
- **功能**: 写入文件内容
- **参数**: file_path (路径), content (内容), append (追加模式，默认 False)
- **特性**: 自动创建父目录，支持追加模式

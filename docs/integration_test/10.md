### 🟡 中优先级

**1. `/search` 命令导入错误**
- **位置**: `src/tui_chatbot/frontend.py:517`
- **现象**: 执行 `/search` 显示 "[Error: attempted relative import beyond top-level package]"
- **原因**: 使用了错误的相对导入 `from ..search import`
- **修复建议**: 改为 `from .search import` 或 `from tui_chatbot.search import`
- **影响**: 搜索功能完全不可用
- **复现**: 100%

```python
# 当前（错误）
from ..search import MessageSearchEngine, SearchScope

# 修复后
from .search import MessageSearchEngine, SearchScope
```

## 3. 缓存功能测试

### 3.1 测试内容

验证 CachedSearchEngine 的缓存机制：
- 首次索引缓存未命中
- 重复索引缓存命中
- 会话更新后缓存失效
- 搜索功能正常

### 3.2 测试代码

```python
from tui_chatbot.search import CachedSearchEngine
from tui_chatbot.session import ChatSession, SessionMetadata
from tui_chatbot.agent.types import UserMessage, AssistantMessage, TextContent
from datetime import datetime

# 创建测试会话
metadata = SessionMetadata(
    id='test123',
    title='Test Session',
    created_at=datetime.now(),
    updated_at=datetime.now(),
    model='gpt-4'
)
session = ChatSession(metadata=metadata)
session.messages.append(UserMessage(content='Hello world'))
session.messages.append(AssistantMessage(content=[TextContent(text='Hi there!')]))
session.messages.append(UserMessage(content='How are you?'))

engine = CachedSearchEngine()

# 第一次索引（应该 miss）
engine.index_session(session)
stats1 = engine.get_cache_stats()

# 清空索引但保留缓存，第二次索引（应该 hit）
engine.clear_index()
engine.index_session(session)
stats2 = engine.get_cache_stats()

# 验证缓存命中
assert stats2['hits'] == 1
assert stats2['misses'] == 1

# 测试缓存失效
session.metadata.updated_at = datetime.now()
engine.clear_index()
engine.index_session(session)
stats3 = engine.get_cache_stats()
assert stats3['misses'] == 2
```

from tui_chatbot.frontend import Frontend
from tui_chatbot.core.events import AgentEvent, AgentEventType
from tui_chatbot.indicator import StreamingIndicator

frontend = Frontend(MockDaemon())

# 初始状态
assert frontend._indicator is None

# 启动指示器

frontend.render_event(event_start)
assert isinstance(frontend._indicator, StreamingIndicator)

# 内容 token 更新
event_token = AgentEvent(type=AgentEventType.CONTENT_TOKEN, data='Hello')
frontend.render_event(event_token)

# 停止指示器
event_end = AgentEvent(type=AgentEventType.MESSAGE_END)
frontend.render_event(event_end)
assert frontend._indicator is None

## 4. 发现的问题

### 4.1 高优先级 - 已修复

**问题 1**: AgentEventType 重复定义导致事件类型比较失败
- **原因**: `core.events.AgentEventType` 和 `agent.types.AgentEventType` 是两个不同的枚举类
- **修复**: 统一从 `core.events` 导入
- **状态**: ✅ 已修复

**问题 2**: AgentEvent Pydantic 模型缺少 partial_result 字段
- **原因**: `extra="forbid"` 导致无法设置 partial_result
- **修复**: 添加字段并改为 `extra="ignore"`
- **状态**: ✅ 已修复

**问题 3**: ChatResult Pydantic 模型验证失败
- **原因**: content 字段必需但未提供，messages 字段类型不匹配
- **修复**: content 设为可选，messages 接受 Any 类型
- **状态**: ✅ 已修复

### 4.2 中优先级 - 已修复

**问题 4**: TPS 统计时间计算异常
- **现象**: 显示 1775573227.7s 这样的巨大数字
- **原因**: `asyncio.get_event_loop().time()` 和 `time.time()` 时间基准不同
- **修复**: 统一使用 `time.time()`
- **状态**: ✅ 已修复

**问题 5**: 缺少 __main__.py 文件
- **现象**: 无法使用 `python -m tui_chatbot`
- **修复**: 已添加 `src/tui_chatbot/__main__.py`
- **状态**: ✅ 已修复
